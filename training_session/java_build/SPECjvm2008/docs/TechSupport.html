SPECjvm2008 Tech Support

Version 1.0
Last modified: March 23, 2008

The SPECjvm2008 benchmark is released as-is, with no expectation of support or warranties given.

The benchmark, including source code for the benchmark, is available free of charge.

A support email alias for questions and bug reports exists. Before mailing the alias, please carefully read the Known Issues document, the User Guide, the FAQ and the Run Rules. The email alias is jvm2008support@spec.org. This email alias is supported by the java group at SPEC.
